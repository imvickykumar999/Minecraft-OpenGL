> ![image](https://github.com/imvickykumar999/Tesseract-Image-Search/blob/main/static/loading-cat.gif?raw=true)

```python
python3 -m venv env
env\Scripts\activate
pip install --upgrade pip

pip install -r requirements.txt
python main.py
deactivate
```
